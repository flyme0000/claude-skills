# 代码审查规则

仅通过 **`commit/SKILL.md`** 引用，不单独使用。

---

## 检查项

对 staged diff 新增行逐行匹配以下规则。命中后在源文件问题行上方插入 `TODO))` 标记注释。

### Error 级（阻断提交）

| 规则     | 检测目标                                            | 说明                                                          |
|--------|-------------------------------------------------|-------------------------------------------------------------|
| 动态执行   | `eval()`、`exec()` 调用                            | 任意位置出现即命中                                                   |
| SQL 注入 | f-string 中含 SQL 关键字 + `{}` 插值；字符串拼接 SQL 关键字     | `select`/`insert`/`update`/`delete`/`drop`/`alter`/`create` |
| 不安全请求  | `verify=False`                                  | 跳过 TLS 证书验证                                                 |
| 密钥硬编码  | `api_key`/`password`/`secret`/`token` 赋值为字符串字面量 | 值长度至少 4 字符                                                  |

### Critical 级（阻断提交）

| 规则      | 检测目标                                                   | 说明                      |
|---------|--------------------------------------------------------|-------------------------|
| 调试残留    | `print()`、`pdb.set_trace()`、`console.log()`、`debugger` | 行首 `print()` 或任意位置出现其余项 |
| 命令注入    | `shell=True`                                           | subprocess 调用中的非安全模式    |
| 不安全反序列化 | `pickle.load()` / `pickle.loads()`、`yaml.load()`       | 可能导致远程代码执行              |

### Warning 级（不阻断）

| 规则    | 检测目标                                            | 说明              |
|-------|-------------------------------------------------|-----------------|
| 阻塞调用  | `time.sleep()`                                  | 生产代码中应避免固定延时    |
| 子进程调用 | `subprocess.Popen`、`subprocess.run`、`os.system` | 需确认是否可替换为安全替代方案 |

### Info 级（不阻断）

| 规则   | 检测目标                        | 说明             |
|------|-----------------------------|----------------|
| 待办标记 | `TODO`、`FIXME`、`XXX`、`HACK` | 仅标记新增行中已存在的待办项 |

---

## 标记注释格式

```python
# TODO)) [级别] @review 分类: 描述
# TODO)) [error] @review 不安全代码: eval() 调用
# TODO)) [critical] @review 调试残留: print() 调用
# TODO)) [warning] @review 阻塞调用: time.sleep()
# TODO)) [info] @review 待办标记: TODO
```

注释符号根据文件扩展名自动适配（`#`、`//`、`<!-- -->` 等）。

---

## 标记管理

- 标记**随代码入库**，进入版本控制
- CI 可扫描 `TODO)) [error] @review` 拦截未修复问题
- `grep "TODO))"` 可列出所有自动审查标记
- 修复问题后由开发者手动删除标记行

---

## 质量检查清单

- [ ] 所有检查项按级别分组（error / critical / warning / info）
- [ ] error 和 critical 级问题阻断提交（退出码 1）
- [ ] warning 和 info 级问题不阻断提交（退出码 0）
- [ ] 标记注释格式为 `TODO)) [级别] @review 分类: 描述`
- [ ] 注释符号根据文件扩展名自动适配
- [ ] 多条标记合并为连续注释块，不重复插入
